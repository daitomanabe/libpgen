

# Packet class

## Description

This class provides packet functionality.
It has protocol specialized header classes. 
Developer can extend new protocol packet class.

This class has packet's raw data. 
That is implemented with pgen::types::data_container class.
This class excutes header::write() and heade::read(),
to write/read datas to/from data_container.
