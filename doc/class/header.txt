


# Header Class

## Description

This class provides protocol header functionality.
Developer can implement specific header class, 
that is inherrited from header class.


## Member Infomation
 
 + static size_t min_length, minimum header length
 + static size_t max_length, maximum header length

 + void write(void*, size_t), write header binnary to memory.
 + void read(const void*, size_t), read header binnary from memory.
