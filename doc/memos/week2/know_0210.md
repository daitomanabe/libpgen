### c++ 正規表現

```
[:]([0-9a-f]{1,2}|[0-9a-f]{1,2})[:]
```

```
std::string str = ""; 
std::regex re("正規表現");
std::smatch match; // string型の文字列に対するマッチング結果を格納

/*
 * bool regex_match(std::string, std::smatch, std::regex);
 */
if (regex_match(str, match, re)) { 
	for (int i=0; i<match.size(); i++) {
		std::cout << match[i] << ":";
	}
	std::cout << std::endl;
}

```
